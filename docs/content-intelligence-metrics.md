# 内容竞争情报指标规则（P0）

公式版本：`content-metrics.p0.1`

## 通用规则

- 缺失值保持 `null`，不按 0 填充。
- 比例指标在 `view <= 0` 时返回 `null`。
- 每个指标保存 `sample_size`、时间窗口、分子、分母和 Evidence ID。
- 样本数少于 3 时保留可计算值，但显示小样本警告；爆款率样本数少于 3 时返回 `null`。
- 原始展示值不做缩尾。评分归一化可以截断极端值，但必须记录评分版本。
- 第一轮快照不计算增长率。
- 确定性实现位于 `src/intelligence/metrics.py`；所有结果使用固定 `MetricName`，并要求至少一个 Evidence ID。
- `sample_size` 表示进入该指标审计范围的样本数；如果有效分母排除了 `uncertain` 或缺失字段，则以 `denominator` 单独记录真实分母。

## 公式

| 指标 | 公式 | 窗口与缺失规则 |
|---|---|---|
| 互动率 | `(点赞+投币+收藏+评论+分享) / 播放` | 任一分子字段缺失或播放不大于 0 时为 `null` |
| 收藏率 | `收藏 / 播放` | 收藏缺失或播放不大于 0 时为 `null` |
| 投币率 | `投币 / 播放` | 投币缺失或播放不大于 0 时为 `null` |
| 评论率 | `评论 / 播放` | 评论缺失或播放不大于 0 时为 `null` |
| 发布频率 | `30 天内相关投稿数 / 30 * 7` | 单位为条/周；`sample_size` 为30天内实际审计投稿数；Creator 近期样本缺失时为 `null` |
| 播放中位数 | 有效样本 `view` 中位数 | `view=0` 可参与；缺失值排除 |
| 互动中位数 | 每条有效样本互动总数的中位数 | 任一互动字段缺失的样本排除 |
| 爆款率 | `播放 >= max(样本播放中位数*3, 10000)` 的视频数 / 有效样本数 | 有效样本少于 3 时为 `null` |
| 相关内容占比 | `relevant / (relevant + irrelevant)` | `uncertain` 不进入分母并单独展示 |
| 样本覆盖率 | `实际代表视频数 / 6` | 最大为 1；不足时展示原因 |
| 搜索可见度 | `该创作者去重视频数 / 本次全部去重视频数` | 只代表当前搜索快照 |
| 样本份额 | `该创作者有效样本播放总和 / 入选竞品有效样本播放总和` | 必须写“样本份额”，禁止写“市场份额” |

手算回归覆盖：单视频互动率/收藏率/投币率/评论率、30天发布频率、中位数缺失排除、爆款阈值与小于3条规则、`uncertain` 不进入相关占比分母、覆盖率上限、搜索可见度和样本份额。

## 评分与指标分工

竞品评分用于选择 Top 5，指标用于解释已选竞品，两者不可混写。评分权重版本为 `competitor-score.p0.1`，LLM 只允许输出语义相关性标签；候选数量、活跃度、互动、扣分、总分和排序由程序计算。

评测中的 Retrieval Recall 只以账号级 `qualified_reference` 为分母。搜索视频相关、`discovery_only` 和 `emerging_candidate` 只用于发现与诊断，不计入 Recall。
