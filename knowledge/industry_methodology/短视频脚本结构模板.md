# 短视频脚本结构模板

> 来源：综合整理自行业运营方法论与实战经验 | 整理日期：2025年

---

## 一、通用结构（黄金公式）

### 1.1 开头（0-3秒）：钩子

- **目标**：让用户停下来
- **方法**：
  - 制造悬念（"接下来发生的事……"）
  - 提出问题（"为什么你的XX总是XX？"）
  - 展示结果（先给效果，再讲方法）
  - 抛出数据（"90%的人都不知道……"）

### 1.2 铺垫（3-15秒）：背景

- **目标**：建立期待
- **方法**：
  - 说明为什么要做这个内容
  - 暗示用户看完能获得什么价值
  - 建立与用户的关联（"你有没有遇到过……"）

### 1.3 核心（15-50秒）：内容

- **目标**：传递价值
- **方法**：
  - 主体内容，每3-5秒一个信息点
  - 使用数字和清单结构（"第一……第二……第三……"）
  - 用案例和故事支撑观点
  - 保持节奏紧凑，删除无效内容

### 1.4 效果（50-65秒）：展示

- **目标**：强化说服
- **方法**：
  - 前后对比效果展示
  - 数据佐证
  - 用户评价或反馈

### 1.5 结尾（65-75秒）：引导

- **目标**：促进行动
- **方法**：
  - 总结要点（一句话概括核心价值）
  - 引导互动（"觉得有用就点个赞"）
  - 设置悬念引导关注（"下期更精彩"）

## 二、不同时长的脚本结构

### 2.1 15秒短视频

```
0-3秒：钩子（悬念/冲突/提问）
3-12秒：核心内容（单一信息点，快速传递）
12-15秒：引导互动（一句话收尾）
```

**适用场景**：单一观点、快速展示、情绪表达

### 2.2 30秒短视频

```
0-3秒：钩子（悬念/冲突/提问）
3-10秒：铺垫（背景/痛点）
10-25秒：核心内容（2-3个信息点）
25-30秒：引导互动（收尾+互动）
```

**适用场景**：简单教程、观点表达、产品展示

### 2.3 60秒短视频

```
0-3秒：钩子（悬念/冲突/提问）
3-15秒：铺垫（背景/痛点/价值预告）
15-45秒：核心内容（3-5个信息点）
45-55秒：效果展示（前后对比/数据佐证）
55-60秒：引导互动（总结+互动+关注引导）
```

**适用场景**：详细教程、产品测评、知识分享

### 2.4 90秒短视频

```
0-3秒：钩子（悬念/冲突/提问）
3-20秒：铺垫（背景/痛点/价值预告）
20-60秒：核心内容（5-7个信息点，分层递进）
60-80秒：效果展示（前后对比/案例佐证）
80-90秒：引导互动（总结+互动+下期预告）
```

**适用场景**：深度教程、详细测评、故事叙述

### 2.5 3分钟+中长视频

```
0-5秒：钩子（强悬念/震撼信息）
5-20秒：铺垫（背景/痛点/价值预告）
20-120秒：核心内容（分章节，每章节20-30秒）
120-160秒：效果展示/总结
160-180秒：引导互动+下期预告
```

**适用场景**：深度知识分享、长教程、故事类内容

## 三、不同类型内容的脚本结构

### 3.1 教程类

```
Hook（0-3秒）：展示最终效果/成果
问题（3-8秒）：提出用户痛点
步骤一（8-20秒）：第一个关键步骤
步骤二（20-35秒）：第二个关键步骤
步骤三（35-45秒）：第三个关键步骤
总结（最后5秒）：回顾要点 + 引导收藏
```

### 3.2 测评类

```
Hook（0-3秒）：展示产品 + 制造悬念
痛点（3-10秒）：用户最关心的问题
测评（10-60秒）：多维度展示和测试
结论（60-90秒）：明确推荐建议
引导（最后5秒）：引导互动和关注
```

### 3.3 故事类

```
Hook（0-3秒）：抛出最大冲突/悬念
背景（3-10秒）：交代人物和情境
发展（10-30秒）：推进情节
高潮（30-45秒）：转折或高潮
结局（最后5秒）：收尾 + 引导评论
```

### 3.4 观点类

```
Hook（0-3秒）：抛出争议性观点
论据一（3-15秒）：第一个支撑论据
论据二（15-30秒）：第二个支撑论据
论据三（30-40秒）：第三个支撑论据
总结（最后5秒）：强化观点 + 引导讨论
```

### 3.5 盘点类

```
Hook（0-3秒）：预告盘点内容
第1名（3-10秒）：最强/最差的一个
第2名（10-20秒）：次级内容
第3名（20-30秒）：继续盘点
第4-5名（30-45秒）：快速带过
总结（最后5秒）：个人观点 + 引导互动
```

## 四、脚本写作技巧

### 4.1 语言口语化

- 不要书面语，像和朋友聊天一样
- 使用短句，避免长句和复杂句式
- 适当使用语气词增加亲切感

### 4.2 节奏紧凑

- 不要废话，每句话都有信息量
- 删除所有"嗯""啊""那个"等无效词汇
- 每3-5秒一个信息点或画面变化

### 4.3 互动设计

- 在视频中设置互动点（投票、选择、挑战）
- 提问引导评论（"你觉得呢？"）
- 设置悬念引导关注（"下期更精彩"）

### 4.4 视觉配合

- 画面与文字同步
- 关键信息用字幕强调
- 每3-5秒切换一次画面或角度
- 使用BGM卡点增强节奏感

### 4.5 情绪控制

- 语速偏快但清晰
- 关键信息加重语气
- 重要观点前后适当停顿
- 情绪有起伏，避免平淡

## 五、脚本写作SOP

```
1. 确定选题和目标受众
2. 撰写Hook（前3秒）
3. 搭建内容框架（开头-中间-结尾）
4. 填充具体信息点（每3-5秒一个）
5. 设计互动引导（结尾部分）
6. 检查节奏和信息密度
7. 口语化润色
8. 计算时长，调整结构
```
