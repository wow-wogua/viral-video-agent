# 短视频爆款选题公式

> 来源：综合整理自行业运营方法论与实战经验 | 整理日期：2025年

---

## 一、黄金3秒法则

根据巨量引擎数据实验室统计，用户平均滑动速度已缩短至1.2秒/视频。前3秒的标题和画面决定了80%的用户留存率。

**核心要点**：
- 前3秒必须抛出核心冲突、悬念或价值承诺
- 避免平淡的自我介绍或背景铺垫
- 语速偏快，信息密度高
- 画面要有冲击力或吸引力

---

## 二、三大核心公式

### 2.1 悬念公式：信息缺口 + 情绪钩子

制造用户不知道的信息缺口，激发好奇心。

**公式结构**：
```
你不知道的/没想到的 + [具体领域] + [悬念暗示]
```

**示例**：
- "月薪3千的我，如何穿出月薪3万的感觉？"
- "为什么你的底妆总是卡粉？"
- "这个方法，99%的人都不知道"
- "你以为的XX其实是XX"

### 2.2 对比公式：反差 + 冲突

通过A vs B的对比制造冲突，吸引用户停留。

**公式结构**：
```
[低成本/简单方案] vs [高成本/复杂方案] + [效果对比]
```

**示例**：
- "29块平替 vs 399块大牌粉底液"
- "用100块做出了1000块的效果"
- "花3小时做的事，我只用3分钟"
- "普通人 vs 高手的区别就在这一步"

### 2.3 结果前置公式：直接展示效果

先展示最终效果或成果，再讲方法，用结果吸引用户。

**公式结构**：
```
用这个方法/技巧 + [具体成果数据] + [暗示你也能做到]
```

**示例**：
- "用这个方法，7天涨粉10万"
- "看完这个视频，你会发现自己之前的眼妆都画错了"
- "10分钟搞定一道硬菜，老公吃了直呼内行"
- "3个月从0到10万粉，我只做了这一件事"

## 三、选题框架（5W1H模板）

确定选题时，用以下框架自检：

| 维度 | 问题 | 说明 |
|------|------|------|
| **Why** | 为什么做这个选题？ | 用户痛点/热点需求/信息差 |
| **Who** | 目标受众是谁？ | 明确年龄、性别、兴趣画像 |
| **What** | 内容核心是什么？ | 一句话说清视频价值 |
| **When** | 什么时候发布？ | 用户活跃高峰期（12-14点、18-22点） |
| **Where** | 在哪个平台发布？ | 抖音/快手/B站/小红书 |
| **How** | 怎么呈现内容？ | 口播/剧情/教程/vlog |

## 四、8种爆款文案公式

| 序号 | 公式类型 | 核心逻辑 | 适用场景 |
|------|----------|----------|----------|
| 1 | **痛点提问式** | 直接戳中用户痛点 | 教程、避坑类 |
| 2 | **结果前置式** | 先展示效果再讲方法 | 教程、测评类 |
| 3 | **争议/反常识式** | 打破用户认知 | 观点、科普类 |
| 4 | **数字量化式** | 用具体数字吸引注意 | 盘点、攻略类 |
| 5 | **对比式** | A vs B 制造冲突 | 测评、平替类 |
| 6 | **故事式** | 用叙事吸引用户 | 情感、经历类 |
| 7 | **教程式** | 提供具体方法 | 技能、工具类 |
| 8 | **清单式** | 罗列要点，信息密度高 | 推荐、攻略类 |

## 五、选题自检清单

确定选题前，用以下清单逐项检查：

- [ ] 目标用户是谁？他们关心什么？
- [ ] 能否一句话说清这条视频的价值？
- [ ] 有没有情绪触发点（好奇/共鸣/争议/实用）？
- [ ] 与账号定位是否一致？
- [ ] 是否有足够的信息增量（不是老生常谈）？
- [ ] 标题是否有吸引力？能否让用户停下来？
- [ ] 封面是否清晰传达了内容主题？

## 六、不同行业的选题公式应用

| 行业 | 痛点方向 | 公式应用示例 |
|------|----------|-------------|
| 美妆 | 花钱多/踩雷/不会选 | "29块平替 vs 399块大牌" |
| 美食 | 不会做/做不好/想尝新 | "10分钟搞定一道硬菜" |
| 健身 | 坚持难/方法错/效果差 | "用这个方法，30天瘦10斤" |
| 职场 | 面试难/升职慢/薪资低 | "当了10年HR，我告诉你面试真相" |
| 科技 | 不会选/怕踩雷/想省钱 | "XX元价位最值得买的XX" |
| 教育 | 学不会/效率低/方法错 | "90%的人不知道的学习方法" |
